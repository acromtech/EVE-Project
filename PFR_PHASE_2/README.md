# PFR - Phase 2

## Prérequis
**GCC**
```
sudo apt install gcc
```
**Make**
```
sudo apt install make
```
**jdk 19**
```
sudo apt install openjdk-19-jdk
```
**Maven**
```
sudo apt install maven
```
**Tree (Optionnel)**
```
sudo apt install tree
```
